# Practical 2 – POS Tagging and Lemmatization using NLTK

## Aim

To analyze textual data using NLTK by performing Part-of-Speech (POS) tagging and converting words into their meaningful base forms using lemmatization.

## Objectives

- Perform Part-of-Speech (POS) tagging on text.
- Understand common POS tags used in NLTK.
- Apply lemmatization to convert words into their base forms.
- Compare original words with their lemmatized forms.

## Description

This practical demonstrates how to analyze textual data using the NLTK library. It covers Part-of-Speech (POS) tagging to identify the grammatical role of each word in a sentence and lemmatization to convert words into their meaningful dictionary forms. These techniques are essential preprocessing steps in many Natural Language Processing applications such as text classification, information extraction, and sentiment analysis.

## Learning Outcomes

After completing this practical, you will be able to:

- Perform POS tagging using NLTK.
- Interpret common POS tags.
- Apply lemmatization using WordNetLemmatizer.
- Prepare text for further NLP tasks.
