# Practical 1 – Text Preprocessing using NLTK

## Aim

Perform text preprocessing tasks using the NLTK library in Python.

## Topics Covered

- Sentence Tokenization
- Word Tokenization
- Stop Word Removal
- Stemming

## Libraries Used

- Python
- NLTK

## Files Included

- Practical-01.ipynb
- TextBlob-Learning.ipynb (Additional Practice)
- README.md

## Description

This practical demonstrates the basic text preprocessing techniques used in Natural Language Processing (NLP). It also includes additional TextBlob examples to further explore NLP concepts beyond the practical requirements.
